# 文件上传

使用 antd@3 的 Upload 组件封装的上传组件

<!-- <code src="./a.tsx"></code> -->
<!-- <code src="./b.tsx"></code> -->

```jsx
import { Upload } from 'suid-supplement';

export default () => <Upload dragPower />;
```

支持[antd3:Upload](https://3x.ant.design/components/upload-cn)组件

| 属性 | 描述 | 类型     | 默认值   |
| ---- | ---- | -------- | -------- |
| 小明 | 25   | `工程师` | `工程师` |
| 小红 | 22   | 设计师   | 设计师   |
| 小刚 | 28   | 营销专员 | 营销专员 |
